Welcome to HapD3cation! This application is a prototype for automated generation of 3D haptification of information visualization.
... a lot of ation's, right?

It is created in the manner of Vega-Lite, but in 3D. Therefore it uses X3DOM to render the threedimensional scenes and D3 to bind data to DOM-elements and to dynamically manipulate the DOM.

The example data is from Our Wold in Data. The following examples mainly use the current covid-19 vaccination data.
